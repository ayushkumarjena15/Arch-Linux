### 🛠 Arch Linux: Installation Guide
Arch Linux is a lightweight and flexible Linux® distribution that tries to Keep It Simple (KISS). Follow this streamlined guide to deploy the core system:

#### 1. Setup Environment
- **Scan for WiFi**: `iwctl`
- **Synchronize Clock**: `timedatectl set-ntp true`
- **Partitioning**: `cfdisk /dev/sda`

#### 2. Format & Mount
- **Format Root**: `mkfs.ext4 /dev/sda2`
- **Mount**: `mount /dev/sda2 /mnt`

#### 3. Core Installation
- **Install Base**: `pacstrap /mnt base linux linux-firmware vim`
- **Generate Fstab**: `genfstab -U /mnt >> /mnt/etc/fstab`

#### 4. System Chroot
- **Enter Chroot**: `arch-chroot /mnt`
- **Set Timezone**: `ln -sf /usr/share/zoneinfo/Region/City /etc/localtime`
- **Generate Locale**: `locale-gen`
- **Hostname**: `echo "arch-system" > /etc/hostname`

Quick commands for daily system maintenance:

| Command | Description |
